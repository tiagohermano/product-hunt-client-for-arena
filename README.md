## About this project

This is a Frontend technical assesment for Arena.im

## Getting Started

First, run the development server:

```bash
cp .env.example .env # and paste in the PRODUCT_HUNT_TOKEN environment variable into the .env file
pnpm install
pnpm dev
```

## Tests

To run the test suite

```bash
pnpm test
```

## Tech Stack

### Frontend

- Next.js
- Typescript
- TailwindCSS
- Shadcn/ui

### Testing
